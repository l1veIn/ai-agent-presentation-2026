# AI 智能体科研工作流演示 (2026) 🦞

一份基于 Slidev 制作的演示文稿，简要梳理 2026 年初 AI 智能体（Agents）的发展脉络。内容涵盖了 IDE 工具链的演进、OS 级控制台的探索，以及国内围绕 OpenClaw 展开的“龙虾大战”生态切片。

希望能以相对直观的方式，探讨现代工作流如何从“对话式建议”逐步过渡到“闭环自动执行”。

## 🤖 关于本项目的制作

> **注：** 本项目也是一次验证 AI 工程能力的实践。本幻灯片的主要工作——包括大纲解构、内容迭代、Slidev 代码排版、样式调整以及文内的配套概念插图，皆由 AI 编程代理（Antigravity）辅助或全自动生成。人类在此过程中主要负责提示词输入、方向把控与最终 Review。

## 💻 本地运行与演示

如果您需要将此源码克隆到本机进行演示或二次修改：

```bash
# 1. 安装依赖
npm install

# 2. 启动本地预览服务
npm run dev
```

运行成功后，在浏览器中打开 `http://localhost:3030` 即可。
