# 设计风格指南

## 整体风格：暗色科技风 + 渐变高光

- **背景**：深蓝黑（#0f0f1a），微妙的网格纹理
- **主色调**：蓝紫渐变 `linear-gradient(135deg, #667eea, #764ba2)`
- **强调色**：青绿色 `#00d2ff`（用于高亮、跃迁、重点）
- **警告色**：橙红渐变（用于复杂度雪崩曲线等警示图表）
- **字体**：中文思源黑体，英文 Inter / JetBrains Mono（代码）

## 关键视觉元素

| 页面类型 | 视觉语言 |
|---------|---------|
| 路线图 | 发光路径 + 节点动画 |
| 理论彩蛋 🎓 | 底部渐变分割线 + 彩蛋 icon + 特殊背景色 |
| 实战演示 | 终端/编辑器截图风格全屏 |
| 对比表格 | 暗色卡片布局，无传统表格线 |
| 数据可视化 | 带渐变的图表，动画绘制 |
| 龙虾大战 | 各大厂 logo 战场全景布局 |

## 动画建议

- 路线图节点逐个亮起
- 控制论框图：先出开环 → 反馈线动画亮起 → 变闭环
- 复杂度雪崩曲线：逐步绘制，每加一个节点下降一格
- 三论回顾大图：三个模块依次淡入
