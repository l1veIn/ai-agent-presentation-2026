# 第六章：垂直工具 & 工作流设计 (4-5 页)

> **叙事定位**：从"深度"转向"广度"。前五章按纵向演进（对话→工具→电脑操控），这一章横向展开，介绍各垂直领域的专业工具，并引导听众设计自己的工作流组合。

## 6.1 垂直领域 AI 工具矩阵

> 用暗色卡片式布局展示，每张卡片包含：工具名、一句话定位、典型使用场景、推荐指数。

### 📚 文献研究

**NotebookLM (Google)**
- 一句话：把论文 PDF 喂给它，然后和论文对话
- 核心能力：基于文档的 AI 对话、自动生成音频摘要（播客模式）、多文档交叉引用
- 适合场景：论文精读、文献综述、快速理解新领域
- 对 AI 学院学生的特殊价值：可以把多篇论文一起喂进去，让它帮你找 cross-paper insights

### 🎨 设计创作

**Lovart**
- 一句话：AI 驱动的设计工具，从描述到设计稿
- 核心能力：根据自然语言生成设计方案、支持多轮迭代
- 适合场景：论文配图、海报制作、PPT 视觉设计、实验流程图
- 和 Midjourney/DALL-E 的区别：更偏向排版设计而非单纯图片生成

### 💻 代码协作

**GitHub Copilot**（回顾式提及）
- 一句话：最轻量的 IDE AI 补全
- 适合场景：已经在用 VS Code、不想换编辑器的同学
- Agent 模式补充：2026 年的 Copilot 已经不只是补全了

### 📊 数据分析

**Julius AI / ChatGPT Code Interpreter**
- 一句话：上传数据，自然语言做分析
- 核心能力：自动选择统计方法、生成可视化、导出报告
- 适合场景：实验数据处理、统计分析、快速出图
- 和手写 Python 的对比：速度快 10 倍，但可控性低

### ✍️ 写作辅助

**Grammarly / WritingMate**
- 一句话：英文论文润色的最后一道防线
- 核心能力：语法纠错、风格优化、学术写作规范检查

### 🇨🇳 国产 AI 垂直工具 — 零门槛的科研利器

> 对于在校学生来说，国产工具**不需要科学上网**是巨大优势。

**豆包·扣子 (Coze)**
- 一句话：字节跳动的 AI 应用构建平台
- 核心能力：拖拽式搭建 AI Bot、可调用插件、支持工作流编排
- 科研场景：快速搭一个论文摘要 Bot、数据查询 Bot
- 特色：飞书深度集成，适合团队协作

**DeepSeek Artifacts / Chat**
- 一句话：国产推理之王，开源免费
- 核心能力：极强的数学推理和代码能力，R1 思维链可见
- 科研场景：数学推导验证、算法设计、代码 debug
- 特色：完全开源，可本地部署，论文复现利器

**智谱 ChatGLM / 清言**
- 一句话：清华系出品，学术基因最强
- 核心能力：长文本理解、知识图谱、多模态
- 科研场景：中文论文精读、学术概念解释
- 特色：GLM 系列开源模型可用于研究实验

### 🔍 搜索与研究助手

**Perplexity / 秘塔 AI 搜索**
- 一句话：带引用的 AI 搜索引擎
- 核心能力：联网搜索 + AI 总结 + 源链接引用
- 科研场景：快速调研某个话题的最新进展、找论文、查 API 文档
- 对比：Perplexity（海外源强）vs 秘塔（中文源强）

## 6.2 如何选择？— 工作流组合思维

> 这一页的核心是让听众理解：**不是选"最好的工具"，而是设计"最匹配的工作流组合"**。

**四种典型科研工作流**：

### 工作流 A：实验型研究者（做实验 → 分析数据 → 写论文）
```
NotebookLM        →  Julius AI        →  Claude (Web)     →  Grammarly
(文献调研+精读)      (数据分析+出图)      (论文写作辅助)       (润色终检)
```

### 工作流 B：系统型开发者（写代码 → 搭系统 → 写论文）
```
Claude Code       →  Antigravity     →  OpenClaw          →  NotebookLM
(架构设计+原型)      (核心开发+调试)     (部署+跨应用集成)     (文献补充)
```

### 工作流 C：理论型研究者（读论文 → 推公式 → 写论文）
```
NotebookLM        →  ChatGPT/Claude  →  Lovart            →  Grammarly
(大量文献精读)       (推理验证+写作)     (论文配图+公式图)      (润色终检)
```

### 工作流 D：全国产方案 🇨🇳（零科学上网依赖）
```
秘塔 AI 搜索      →  DeepSeek R1     →  智谱清言           →  豆包·扣子
(文献调研+快速搜索)   (推理+代码实现)     (论文精读+写作)       (搭建自动化 Bot)
```
- 特别适合：校园网环境、对网络稳定性要求高的场景
- 强调：**这套方案完全免费且不需要任何特殊网络配置**

- **给学生的引导**："你觉得自己偏哪一种？你现在的工作流是什么？能不能在某个环节引入一个新工具？"
- 强调：**从一个工具开始，逐步扩展，不要一次全换**
